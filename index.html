How to use
  • A–Z — type a letter into the focused tile; advances to the next empty tile automatically
  • Backspace — clear the current tile; if already empty, move focus to the previous tile
  • ← / → — move focus left or right within the row
  • Space — cycle the focused tile's color (gray → yellow → green → gray)
  • Click a filled tile — same color cycle as Space
  • Enter — submit the row when all 5 tiles are filled; otherwise jump to the next empty tile
  • Paste a word — distributes letters across tiles starting from the focused one
  • trash button — clear the current row and remove all rows after it
  • Colors: green = correct position  |  yellow = wrong position  |  gray = not in word
